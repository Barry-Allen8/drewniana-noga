# DREWNIANA NOGA — Landing Page

Single-page landing page for a premium bespoke furniture brand in Bydgoszcz, Poland.

## Tech Stack

- **Framework:** Next.js 15 (App Router, Server Components)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Components:** Shadcn/UI
- **Animations:** Framer Motion, React Three Fiber + Drei
- **Icons:** Lucide React

## Features

- **Hero:** 3D interactive canvas (torus knot + particles) with mouse orbit controls
- **Price Configurator:** Dimensions (width/height) + wood type (Dąb, Jesion, Orzech)
- **Lead Magnet:** Email capture for PDF "5 Sposobów na konserwację drewna"
- **Contact:** Address, phone, and "Zadzwoń Teraz" CTA

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
